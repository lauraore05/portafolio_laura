# 💻 Portafolio Personal — Laura Ore

Portafolio web personal desarrollado con HTML, CSS y JavaScript.

## 🌐 Vista previa

El sitio incluye las siguientes secciones:

- **Hero** — Presentación con efecto typewriter, foto con glow animado y botones de contacto (WhatsApp, CV, GitHub, LinkedIn).
- **Proyectos** — Lista de proyectos destacados con enlaces a demos y código fuente.
- **Sobre Mí** — Breve descripción personal, habilidades representadas con iconos y foto.
- **Contacto** — Enlaces directos a WhatsApp, Email, GitHub y LinkedIn.

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura semántica |
| CSS3 | Estilos, animaciones y diseño responsive |
| JavaScript | Interactividad, efecto typewriter, partículas y cursor personalizado |
| Google Fonts | Tipografías Inter y Space Grotesk |

## 📁 Estructura del proyecto

```
portafolio_laura/
├── index.html      # Página principal
├── styles.css      # Estilos y animaciones
├── script.js       # Lógica e interactividad
├── favicon.svg     # Icono de pestaña
├── CV.pdf          # Currículum en PDF
├── piña.jpg        # Foto de perfil
└── README.md       # Este archivo
```

## 🚀 Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/lauraore05/portafolio_laura.git
   ```
2. Abre `index.html` en tu navegador.

> No requiere instalación de dependencias ni servidor. Es un sitio estático.

## ✨ Características

- Efecto **typewriter** animado en el título principal
- **Partículas** interactivas en el fondo
- **Cursor personalizado** con efecto de seguimiento
- Animaciones de entrada con **scroll reveal**
- Diseño **responsive** adaptado a móviles y escritorio
- Iconos SVG integrados (sin dependencias externas)
